# Previsão de Avaliação de Jogos

Este projeto utiliza modelos de Machine Learning para analisar e prever a nota de avaliação (User Rating) de jogos digitais.
Utilizando um conjunto de dados com características como gênero, preço, plataforma e qualidades técnicas (gráficos, som, história), o script treina e compara dois modelos de regressão: Regressão Linear e Random Forest Regressor.
O objetivo é identificar quais atributos mais influenciam a recepção de um jogo pelo público, avaliando a performance dos modelos através de métricas como R-squared (R2), MAE (Mean Absolute Error) e RMSE (Root Mean Squared Error).
